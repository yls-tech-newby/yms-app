# yms-app

基于uni-app的yms移动端通用应用

## 本地环境编译


### h5版本：
```
npm run dev:h5
```

### 支付宝小程序版本：
```
npm run dev:mp-alipay
```

### 百度小程序版本：
```
npm run dev:mp-baidu
```

### 微信小程序版本：
```
npm run dev:mp-weixin
```

### 头条小程序版本：
```
npm run dev:mp-toutiao
```

## 编译安卓及ios app
使用[HBuilderX](http://www.dcloud.io/hbuilderx.html)编译app
